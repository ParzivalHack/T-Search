### ABOUT :

The first Search Engine for terminal.

### TESTED ON :

* Termux✅
* Linux✅

### REQUIREMENTS :

* w3m [pkg install w3m -y]

### INSTALLATION & USAGE:

* git clone https://github.com/ParzivalHack/T-Search
* cd T-Search 
* bash T-Search.sh

### FEATURES :

* [+] Search on Google!
* [+] Search anonymously on DuckDuckGo!
* [+] Beginner friendly!
* [+] Works on Termux!
